Менеджер аккаунтов для Cristalix. Пока приложение открыто, оно само ловит токены аккаунтов, под которыми ты заходишь через лаунчер, и хранит их зашифрованными. Дальше переключаешься между аккаунтами в один клик — без паролей и повторных входов.

## Что умеет

- **Запуск в один клик** — запустить аккаунт, закрыть клиент, зайти без аккаунта.
- **Импорт по логину и паролю** — даёшь список аккаунтов, приложение само вводит их в лаунчер и сохраняет токены. Видит неверный пароль, пропускает дубли, показывает прогресс и в любой момент отменяется.
- **Обновление токенов** — одной кнопкой перелогинивает аккаунты, у которых токен скоро истекает.
- **Группы** — собираешь аккаунты в группы и запускаешь всю группу разом.
- **Профили настроек** — сохраняешь настройки игры (управление, графика, голосовой чат) и применяешь к аккаунту. Есть готовый профиль «Минимальные настройки» — чтобы запускать пачкой или на слабом железе.
- **Инфо об игроке** — 3D-скин, головы, группа, подписка, онлайн-статус, метки и избранное.

## Как пользоваться

Скачай и запусти `AccountChanger.exe` — откроется окно приложения. Закрываешь окно — оно сворачивается в трей и продолжает ловить токены в фоне. Чтобы выйти полностью, нажми «Закрыть» или сделай это через иконку в трее.

На Linux файлу сначала нужно разрешить запуск — `chmod +x AccountChanger-linux-x86_64`, дальше `./AccountChanger-linux-x86_64`. Ещё понадобится Java для лаунчера.

## Windows и Linux

Windows умеет всё, на Linux — почти всё:

| Возможность | Windows | Linux |
| --- | :---: | :---: |
| Ловля токенов, запуск, группы, профили, инфо об игроке | ✅ | ✅ |
| Импорт по логину и паролю | ✅ | ❌ |
| Обновление токенов | ✅ | ❌ |
| Автоклик «ИГРАТЬ» | ✅ | ❌ |

На Linux то, чего нет, делается вручную: заходи в аккаунт через лаунчер (токен подхватится сам), а вместо автоклика включи «Автовход» в настройках аккаунта.

## Безопасность и приватность

Токены хранятся в зашифрованном виде и привязаны к твоему компьютеру: на Windows — через DPAPI, на Linux — ключом, который доступен только тебе. Приложение отправляет лишь анонимную статистику онлайна и отключается в настройках.
